# proyectoMovil
#El proyecto es un chat en tiempo real con base de datos MySQL
#La carpeta ScriptsPHP son los archivos php que funcionan como conectores
#a la base de datos. Esta carpeta tendría que ir en su servidor.
#En el aplicativo, en la clase links, la variable IP tendrá como nombre la
#dirección ip o el dominio donde se encuentra la carpeta ScriptsPHP. Tener en
#en cuenta si es http o https.
#En caso de que su servidor sea un http, en la carpeta res, en la carpeta xml, hay
# un archivo llamado react_native_config.xml, allí en la direccion 192.168.0.16, 
#elimine esa direccion y en su caso escriba la ip o dominio que ud usa.
